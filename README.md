# 📝 ToDo App

A simple and interactive ToDo Application built using **HTML, CSS, and JavaScript**.

## 🚀 Live Features

✅ Add Tasks

✅ Mark Tasks as Completed

✅ Delete Tasks

✅ Track Total Tasks

✅ Track Completed Tasks

## 📸 Project Overview

This application helps users manage their daily tasks efficiently.

Users can:

* Add new tasks
* Mark tasks as completed
* Delete unwanted tasks
* View total number of tasks
* View completed task count

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
